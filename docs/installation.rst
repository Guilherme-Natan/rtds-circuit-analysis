Installation
============

As a command line application
-----------------------------

If you plan on using the :ref:`command line interface <usage-cli>` for this software, you can use
`pipx <https://github.com/pypa/pipx>`_ for this:


.. code-block:: bash

    pipx install rtds-circuit-analysis

As a python library
-------------------

If you want to use the :ref:`library <usage-lib>`, however, you might prefer using `pip
<https://pip.pypa.io/en/stable/>`_ for the installation:

.. code-block:: bash

    pip install rtds-circuit-analysis
